Chapter 1: The Real and Complex Number System



Summary of Chapter

Notational Conveniences

Rudin has a few notational conveniences early on that are listed as definitions, but are really a mix of notation and definition. They are essentially as follows:

  1. If $A$ is a set, $x \in A$ indicates $x$ is an element of $A$, and $x \not\in A$ indicates $x$ is not an element of $A$.
  2. If a set has no elements, it is the empty set. If it has at least one element, it is nonempty. Rudin gives no special symbol for the empty set, but $\emptyset$ is common here.
  3. If there are sets $A$ and $B$ with every element in $A$ as a member of $B$, then we say $A$ is a subset of $B$ and write $A \subset B$ (or $B \supset A$). (This notation seems to be falling out of favor - I will elaborate soon enough.)
  4. If $A \subset B$ and there is an element of $B$ that is not an element of $A$, then we may say $A$ is a proper subset of $B$.
  5. If $A \subset B$ and $B \subset A$, then $A$ and $B$ are equal with respect to set elements, and we write $A = B$. Otherwise, we write $A \ne B$.

Note: Increasingly, some mathematicians see the use of $A \subset B$ to be ambiguous; some authors use it to indicate that $A$ is a strict subset of $B$ and use $A \subseteq B$ to indicate the possibility of set equality. Other mathematicians are opting for even more explicit notation for the strict subset case: $A \subsetneq B$. Anyone using this notation will generally use $A \subseteq B$ otherwise if $A$ is not necessarily a strict subset. To summarize:

  1. Convention 1: $A \subset B$ indicates the subset relation, and is agnostic as to whether or not this is strict. (Rudin)
  2. Convention 2: $A \subset B$ indicates specifically a strict subset relation, and $A \subseteq B$ indicates otherwise.
  3. Convention 3: $A \subsetneq B$ indicates specifically a strict subset relation, and $A \subseteq B$ indicates otherwise.

My personal preference is Convention 2, since having at least something to indicate strict set containment is handy for some proofs, but I feel like Convention 3 introduces clutter. Other mathematicians have other opinions, some quite strong. I will generally opt for Convention 2 just to have the notational convenience.


Ordered Sets

In this book and any other text in analysis and topology, set orderings are important structures. Specifically, they allow $ \lt $ and $ \gt $ to be defined as relations and used with all the power that entails. So here's how Rudin defines an order:

Definition 1.5: Let $S$ be a set. An order on $S$ is a relation, denoted by $ \lt $, with the following properties:

  1. If $x \in S$ and $y \in S$, then one and only one of the following statements is true: $x \lt y$, $x = y$, or $y \lt x$.
  2. Given $x, y, z \in S$, if $x \lt y$ and $y \lt z$, then $x \lt z$.

The second property is one called transitivity, which we will see later.

Some other notational conveniences Rudin notes include the ability to write $y \gt x$ in the place of $x \lt y$, and that $x \leq y$ means either $x \lt y$ or $x = y$, making it the negation of $x \gt y$.

Definition 1.6: An ordered set is a set $S$ in which an order is defined.

Rudin has thus far used $\mathbb{Q}$ to indicate the rational numbers - those of the form $\frac{a}{b}$ for integers $a$ and $b$ - and suggesting their structure is not sufficient for our purposes, essentially foreshadowing the construction of the real numbers by pointing out the gaps in $\mathbb{Q}$. However, he does point out we can put the familiar order on $\mathbb{Q}$ by saying $r \lt s$ if and only if $s - r$ is a positive rational number.

Again, Rudin uses convention that is not as specific as some authors like. Mathematicians like to specify sets as either partially or totally ordered. Rudin here seems to be using ordered as synonymous with totally ordered, which is fine for consideration of the rational and real numbers, but not necessarily for order relations on things like sets or graphs.

Definition 1.7: Suppose $S$ is an ordered set and $E \subseteq S$. If there exists a $\beta \in S$ such that $x \leq \beta$ for every $x \in E$, we say that $E$ is bounded above, and call $\beta$ and upper bound of E.

This can all be swapped for the equivalent definitions of bounded below and lower bound where $\beta \leq x$ for every $x \in E$.

Definition 1.8: Suppose $S$ is an (totally) ordered set, $E \subseteq S$, and $E$ is bounded above. Suppose further there exists some $\alpha \in S$ with the following properties:

  1. $\alpha$ is an upper bound of $E$.
  2. If $\gamma \lt \alpha$, then $\gamma$ is not a upper bound of $E$.

    3. Then $\alpha$ is called the least upper bound of $E$, or the supremum of $E$ and we may write $\alpha = \sup E$. Flipping signs as before, we may have the greatest lower bound of $E$, or the infimum of $E$, and write $\alpha = \inf E$.
